# 桃花源维度革新 - 资源需求文档

## 概述
基于Minecraft 1.20.4的数据驱动世界生成系统，重新设计桃花源维度，创建独特的地形、生物群系和结构。

## 已实现的6个生物群系

### 1. 落花寻溪原 (falling_petal_stream)
**诗意描述：** "缘溪行，忘路之远近。忽逢桃花林，落英缤纷"
**特征：**
- 地形：平缓的平原，蜿蜒的忘川溪流
- 氛围：神秘而静谧，花瓣飘落水面
- 颜色：淡粉色天空，浅绿色草地
- 生物：基础动物（羊、猪、鸡、牛）
- 粒子：飘落花瓣效果

**所需方块：**
- `peach_grass_block` - 桃源草方块（已有基础）
- `peach_dirt` - 桃源土（已有基础）
- `peach_stone` - 桃源石材（已有基础）
- `ancient_stone` - 古老石材（已有基础）
- `mossy_ancient_stone` - 苔藓古老石材（已有基础）

### 2. 千耕平畴 (farming_plains)
**诗意描述：** "土地平旷，屋舍俨然，有良田、美池、桑竹之属"
**特征：**
- 地形：开阔的平原，整齐的梯田
- 氛围：安宁祥和，充满生活气息
- 颜色：温暖的田园色调
- 生物：村民、牛、鸡
- 粒子：菌丝效果

**所需方块：**
- `peach_planks` - 桃木板（已有基础）
- `peach_brick` - 桃源砖块（已有基础）
- `peach_tile` - 桃源瓦片（已有基础）
- `white_plaster` - 白色灰泥（已有基础）

### 3. 镜湖百池 (mirror_lakes)
**诗意描述：** "池水清澈如镜，完美倒映着天空与岸边的桃林"
**特征：**
- 地形：点缀在平原之上的梦幻池塘
- 氛围：如梦似幻，水天一色
- 颜色：清澈的水体，梦幻的光影
- 生物：兔子、鸡
- 粒子：菌丝效果

**所需方块：**
- `peach_planks` - 桃木板（已有基础）
- `petal_block` - 花瓣方块（已有基础）

### 4. 叠翠微岚 (verdant_hills)
**诗意描述：** "层层叠叠的绿色丘陵，山间总是缭绕着薄薄的雾气（微岚）"
**特征：**
- 地形：起伏的绿色丘陵
- 氛围：诗意过渡，光影流转
- 颜色：翠绿的植被，朦胧的雾气
- 生物：兔子、鸡
- 粒子：菌丝效果

**所需方块：**
- `peach_planks` - 桃木板（已有基础）
- `peach_brick` - 桃源砖块（已有基础）

### 5. 隐竹之界 (hidden_bamboo_realm)
**诗意描述：** "不足为外人道也"的秘密所在，位于维度边缘的幽深竹林"
**特征：**
- 地形：幽深的竹林，高大的竹子遮天蔽日
- 氛围：神秘感，视野受限
- 颜色：翠绿的竹子，斑驳的光影
- 生物：熊猫、鹦鹉
- 粒子：菌丝效果

**所需方块：**
- `peach_bamboo` - 桃源竹子（已有基础）
- `bamboo_planks` - 竹板（已有基础）
- `bamboo_mat` - 竹席（已有基础）

### 6. 妖灼华原 (enchanting_peach_grove)
**诗意描述：** "桃之夭夭，灼灼其华"，形容此片桃林美到极致，美得超凡脱俗，近乎妖冶"
**特征：**
- 地形：无边无际的粉色海洋
- 氛围：极致美丽，超凡脱俗
- 颜色：粉红色的极致表现
- 生物：基础动物
- 粒子：飘落花瓣效果

**所需方块：**
- `peach_log` - 桃树原木（已有基础）
- `peach_leaves` - 桃树树叶（已有基础）
- `petal_block` - 花瓣方块（已有基础）

- 原版结构生成系统优化

### 🎨 设计建议

1. **保持风格统一**：所有纹理应保持相似的艺术风格
2. **色彩协调**：桃花源以粉色、绿色、白色为主色调
3. **细节丰富**：64x64纹理可以添加更多细节和装饰
4. **材质真实**：考虑不同材质的质感表现
